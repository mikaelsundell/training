Introduction
------------

arri-logc is a set of utility apps for the arri logc wide gamut colorspace.

Documentation
-------------

```shell
mkdir build_cmake &&
cd build_cmake &&
cmake ../ 
    -DCMAKE_PREFIX_PATH=<3rdparty>/build/macosx/arm64.debug 
    -DCMAKE_CXX_STANDARD=17 &&
make
```
